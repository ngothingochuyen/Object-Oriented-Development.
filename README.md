# Object-Oriented-Development.
COURSE OUTLINE

Course Name:	Object-Oriented Development.
 
COURSE DESCRIPTION

This course is designed to provide students with a comprehensive understanding of object-oriented programming (OOP) principles and practices, highlighting their advantages over traditional programming methods. Through an in-depth exploration of fundamental OOP concepts and techniques, students will gain the skills needed to design and develop robust, reusable, and maintainable software.

COURSE OBJECTIVES

The course teaches students the following knowledge and skills:

Basic concepts and essential knowledge for building a computer program using object-oriented programming methods.

Analyzing and designing solutions using object-oriented programming methods based on object-oriented principles and design patterns.

Analytical thinking skills, critical thinking, and systems thinking to solve problems through hands-on lab practice.
